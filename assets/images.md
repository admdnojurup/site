# Image Assets

Naming convention: lowercase, hyphen-separated, prefixed by context.

## Required Images

| File | Description |
|------|-------------|
| `case-crm-dashboard.png` | Screenshot of the Gedarta CRM dashboard showing contacts, proposals, and conversion stats. Used on the case study page (`atvejai.html`). |

| File | Description |
|------|-------------|

| `case-engineering.png` | Engineering case study visual |
| `case-finance.png` | Finance case study visual |
| `case-manufacturing.png` | Manufacturing case study visual |

## Naming Rules

- Prefix with context: `case-`, `client-`, `icon-`, etc.
- Use lowercase with hyphens: `case-crm-dashboard.png`
- Prefer `.png` for screenshots, `.svg` for icons/logos
- All images go in this `assets/` folder
